# Yosoi Docs

Source files for the [Yosoi](https://github.com/CascadingLabs/Yosoi) documentation, served at [cascadinglabs.com/yosoi](https://cascadinglabs.com/yosoi).

## Structure

- **yosoi/** — Core docs: introduction, installation, quickstart, and configuration
- **guides/** — How-to guides covering selectors, multi-item extraction, concurrency, custom types, and more
- **reference/** — Auto-generated API reference (produced by `scripts/generate_api_docs.py` in the Yosoi repo)
